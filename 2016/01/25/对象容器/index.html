Java学习 对象容器

#对象容器

@(Java预习)

##顺序容器
即放进容器中的对象是按照指定的顺序（放的顺序）排列起来的，而且允许具有相同值的多个对象存在。

范型类：import java.util.ArrayList;
private ArrayList<String> test = new ArrayList<String>();

类名：ArrayList
它是一种容器，用来存放 String 的 ArrayList
容器类有两个类型：
容器的类型: e.g. ArrayList
元素的类型: e.g. String
顺序容器，即放进容器中的对象是按照指定的顺序（放的顺序）排列起来的，而且允许具有相同值的多个对象存在

###ArrayList 的操作

  1. xxx.add(string);
  2. xxx.size();
  3. xxx.get(index);
  4. xxx.add(index, string);
  5. xxx.remove(index);
  6. xxx.toArray(string);

##对象数组
所谓对象数组就是String[];
从之前C语言的知识可以知道，这个数组里面每一个元素都是一个字符串的管理者，它指向字符串（java里面没有指针）；

对象数组（区别于基础类型的数组）中的每一个元素其实只是对象的管理者而不是对象本身，因此仅仅创建数组并没有创建其中的每一个对象

用 for-each 循环也是可以遍历的，但如果在循环内调用对象中可以改变对象的成员函数时，将会改变对象数组中元素所管理的对象

###集合容器（Set）

集合（Set）：所有的元素都具有唯一的值，元素在其中没有顺序

导入类库：import java.util.HashSet;
HashSet<String> s = new HashSet<String>();


如图set里放入三个元素，但是有重复，所以只会发现只有两个元素

###Hash 表

Hash 表（又译散列表）：传统意义上的 Hash 表，是能以 int 做值，将数据存放起来的数据结构
Java 的 Hash 表可以以任何实现了 hash() 函数的类的对象做值来存放对象
